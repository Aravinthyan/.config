## .config

# neovim

Install latest version of neovim (master). Install all plugins as follows:

```
:PackerCompile
:PackerInstall
```

# Packages to install

Node.js v12.x

```
curl -fsSL https://deb.nodesource.com/setup_12.x | sudo bash -
sudo apt install -y nodejs
```

Font-Awesome
```
sudo apt install fonts-font-awesome
```
