# nvim

Works on nvim v0.4.4

## Getting started

1) Clone the repo
2) Install all the pre-requiste packages using the pre.sh script
3) Change terminal font to "Source Code Pro for Powerline Semibold", font size 11
4) And run nvim

## vim-plug

This contains a file called plugins.vim, this is where the plugins that are used will be included.

## plug-config

This contians the configs for each of the installed plugins.

## themes

This contains the different configs for the themes that are used with nvim.

## keys

This contains new key mappings.

## general 

This contains the generic settings like set number for nvim.

## init.vim

This calls the other configs in the above directories. This is better and easier to maintain than keeping all the configs in the init.vim file.
