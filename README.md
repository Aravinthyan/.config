# USER

## Installation

```
sudo apt install git
cd ~
rm -rf .*
git clone https://github.com/Aravinthyan/USER.git .
sudo ./setup.sh $USER $HOME
```

## neovim

Install all plugins as follows:

```
:PackerCompile
:PackerInstall
```
